# Microcontroller Training v1
by Bryan A. "CrazyUncleBurton" Thompson  
Last Updated 3/14/2026  

## Overview

These tutorials will allow you to create electric circuits controlled by a computer, and to create programs which have the ability to control the physical world.  

## Prerequisites

We assume that you have some exposure to C/C++ computer programming, and to building basic electrical circuits on a solderless breadboard. 

## Target Platform

M5Stack Tab5 development board w/ESP32 microcontroller

Note:  The libraries are sufficiently generic that the programs will work with other M5Stack controllers and probably with anything running an ESP32 microcontroller.

## Required Software and Accounts

- Microsoft VS Code (Free Download)
- PlatformIO (Free Download)
- Optional:  Microsoft Account for Copilot
- Optional:  GitHub Account

## Lessons

1. Configuring the Integrated Development Environment - Untested
2. Getting Started / Hello World - Untested
3. LCD (Text/Data/Graphics/Touch) - Started
4. Digital Inputs and Outputs - Started
5. Serial Communication (UART / I2C / I2S / SPI / CAN) - Not started
6. Analog to Digital Conversion (ADC) / Digital to Analog Conversion (DAC) / Analog Comparators / Touch - Not Started
7. Interrupts / Timers / Counters / PWM / RMT / Watchdog / MCPWM / Clocks - Not Started
8. Graphical User Interface (LVGL and Squareline Studio) - Not Started
9. Source Control / Creating a New PlatformIO Project - Started
10. Task Scheduler / FreeRTOS - Not Started
11. State Machines - Not Started
12. Debugging - Not Started
13. Testing - Not Started
14. Security (AES/SHA/RSA/ECC/Secure Boot/Flash Encryption/HMAC/RNG)
15. Graphics Subsystem (that may be GUI above)
16. CPU Subsystem (Dual Cores/Vector AI Accel/Interrupt Controller/JTAG)
17. High Speed Peripherals (USB2/Ethernet/SDIO/SDMMC/SPI/LCD/Camera MIPI/CSI)
18. Memory Subsystem (SRAM/Cache/PSRAM/Octal SRAM/MMU)
19. Power Management (Clock Domains / PLL / Power Domains / Sleep Controller)
