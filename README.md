# Microcontroller Training v1
by Bryan A. "CrazyUncleBurton" Thompson  
Last Updated 3/9/2026  

## Overview

These tutorials will allow you to create electric circuits controlled by a computer, and to create programs which have the ability to control the physical world.  

## Prerequisites

We assume that you have some exposure to C/C++ computer programming, and to building basic electrical circuits on a solderless breadboard. 

## Target Platforms

M5Stack Tab5 development board w/ESP32 microcontroller

Note:  The libraries are sufficiently generic that the programs will work with other M5Stack controllers and probably with anything running an ESP32 microcontroller.

## Required Tools

- Microsoft VS Code (Free Download)

## Lessons

5. Interrupts and Timers - Not Started
6. Pulse Width Modulation / Servo and Motor Control - Not Started

8. UART - Not started
9. I2C - Not started
10. SPI - Not Started
11. Graphical User Interface (LVGL and Squareline Studio) - Not Started
12. Source Control - Started
13. FreeRTOS
14. Debugging
15. Testing
